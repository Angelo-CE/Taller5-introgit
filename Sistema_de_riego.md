# Taller5-introgit

# Nombre: Sistema de Riego Automatizado

# Descripción general
El sistema se encarga de medir el nivel del agua de los cultivos, si el nivel de agua es bajo, entonces el sistema abre una válvula para el riego.

# Usuario objetivo: Agricultores 

# Entradas: Medición de humedad de los cultivos.

# Salidas: Riego automatico de los cultivos
